02. Reinforcement Learning

Training Agents for Complex Robotic Tasks​

This chapter provides an in-depth exploration of Reinforcement Learning (RL) techniques specifically tailored for training agents to perform complex robotic tasks. It covers fundamental RL concepts such as states, actions, rewards, and policies, and introduces key algorithms like Q-learning, SARSA, Policy Gradients (REINFORCE), and Actor-Critic methods (A2C/A3C).

Fundamentals of Reinforcement Learning​

Markov Decision Process (MDP)​

An MDP is defined by the tuple (S, A, P, R, γ):

  • S: State space
  • A: Action space
  • P: Transition probability P(s'|s,a)
  • R: Reward function R(s,a,s')
  • γ: Discount factor (0 ≤ γ ≤ 1)

Goal: Find optimal policy π* that maximizes expected cumulative reward:

π* = argmax E[Σ γ^t R_t]

Key Concepts​

Policy (π): Mapping from states to actions

  • Deterministic: a = π(s)
  • Stochastic: a ~ π(·|s)

Value Functions:

  • State value: V^π(s) = E[Σ γ^t R_t | s_0 = s]
  • Action value: Q^π(s,a) = E[Σ γ^t R_t | s_0 = s, a_0 = a]

Bellman Equations:

V(s) = max_a [R(s,a) + γ Σ P(s'|s,a) V(s')]
Q(s,a) = R(s,a) + γ Σ P(s'|s,a) max_a' Q(s',a')

RL Algorithms for Robotics​

Q-Learning (Off-Policy)​

Algorithm:

import numpy as np

class QLearning:
    def __init__(self, n_states, n_actions, alpha=0.1, gamma=0.99, epsilon=0.1):
        self.Q = np.zeros((n_states, n_actions))
        self.alpha = alpha  # Learning rate
        self.gamma = gamma  # Discount factor
        self.epsilon = epsilon  # Exploration rate

    def select_action(self, state):
        if np.random.random() < self.epsilon:
            return np.random.randint(self.Q.shape[1])  # Explore
        return np.argmax(self.Q[state])  # Exploit

    def update(self, state, action, reward, next_state):
        td_target = reward + self.gamma * np.max(self.Q[next_state])
        td_error = td_target - self.Q[state, action]
        self.Q[state, action] += self.alpha * td_error

# Training loop
agent = QLearning(n_states=100, n_actions=4)

for episode in range(1000):
    state = env.reset()
    done = False

    while not done:
        action = agent.select_action(state)
        next_state, reward, done = env.step(action)
        agent.update(state, action, reward, next_state)
        state = next_state

Deep Q-Network (DQN)​

For continuous state spaces:

import torch
import torch.nn as nn
import torch.optim as optim
from collections import deque
import random

class DQN(nn.Module):
    def __init__(self, state_dim, action_dim):
        super().__init__()
        self.network = nn.Sequential(
            nn.Linear(state_dim, 128),
            nn.ReLU(),
            nn.Linear(128, 128),
            nn.ReLU(),
            nn.Linear(128, action_dim)
        )

    def forward(self, x):
        return self.network(x)

class DQNAgent:
    def __init__(self, state_dim, action_dim):
        self.q_network = DQN(state_dim, action_dim)
        self.target_network = DQN(state_dim, action_dim)
        self.target_network.load_state_dict(self.q_network.state_dict())

        self.optimizer = optim.Adam(self.q_network.parameters(), lr=1e-3)
        self.memory = deque(maxlen=10000)
        self.batch_size = 64
        self.gamma = 0.99
        self.epsilon = 1.0
        self.epsilon_decay = 0.995
        self.epsilon_min = 0.01

    def select_action(self, state):
        if random.random() < self.epsilon:
            return random.randint(0, self.q_network.network[-1].out_features - 1)

        with torch.no_grad():
            state_tensor = torch.FloatTensor(state).unsqueeze(0)
            q_values = self.q_network(state_tensor)
            return q_values.argmax().item()

    def store_transition(self, state, action, reward, next_state, done):
        self.memory.append((state, action, reward, next_state, done))

    def train(self):
        if len(self.memory) < self.batch_size:
            return

        batch = random.sample(self.memory, self.batch_size)
        states, actions, rewards, next_states, dones = zip(*batch)

        states = torch.FloatTensor(states)
        actions = torch.LongTensor(actions)
        rewards = torch.FloatTensor(rewards)
        next_states = torch.FloatTensor(next_states)
        dones = torch.FloatTensor(dones)

        # Current Q values
        current_q = self.q_network(states).gather(1, actions.unsqueeze(1))

        # Target Q values
        with torch.no_grad():
            max_next_q = self.target_network(next_states).max(1)[0]
            target_q = rewards + (1 - dones) * self.gamma * max_next_q

        # Loss and optimization
        loss = nn.MSELoss()(current_q.squeeze(), target_q)
        self.optimizer.zero_grad()
        loss.backward()
        self.optimizer.step()

        # Decay epsilon
        self.epsilon = max(self.epsilon_min, self.epsilon * self.epsilon_decay)

    def update_target_network(self):
        self.target_network.load_state_dict(self.q_network.state_dict())

# Training
agent = DQNAgent(state_dim=4, action_dim=2)

for episode in range(1000):
    state = env.reset()
    total_reward = 0

    for step in range(200):
        action = agent.select_action(state)
        next_state, reward, done, _ = env.step(action)

        agent.store_transition(state, action, reward, next_state, done)
        agent.train()

        state = next_state
        total_reward += reward

        if done:
            break

    if episode % 10 == 0:
        agent.update_target_network()

    print(f"Episode {episode}, Reward: {total_reward}, Epsilon: {agent.epsilon:.3f}")

Proximal Policy Optimization (PPO)​

State-of-the-art for robotics:

import torch
import torch.nn as nn
from torch.distributions import Normal

class ActorCritic(nn.Module):
    def __init__(self, state_dim, action_dim):
        super().__init__()

        # Shared feature extractor
        self.shared = nn.Sequential(
            nn.Linear(state_dim, 256),
            nn.ReLU(),
            nn.Linear(256, 256),
            nn.ReLU()
        )

        # Actor (policy)
        self.actor_mean = nn.Linear(256, action_dim)
        self.actor_log_std = nn.Parameter(torch.zeros(action_dim))

        # Critic (value function)
        self.critic = nn.Linear(256, 1)

    def forward(self, state):
        features = self.shared(state)

        # Actor output
        action_mean = self.actor_mean(features)
        action_std = torch.exp(self.actor_log_std)

        # Critic output
        value = self.critic(features)

        return action_mean, action_std, value

    def get_action(self, state):
        action_mean, action_std, value = self.forward(state)
        dist = Normal(action_mean, action_std)
        action = dist.sample()
        log_prob = dist.log_prob(action).sum(dim=-1)
        return action, log_prob, value

class PPO:
    def __init__(self, state_dim, action_dim):
        self.policy = ActorCritic(state_dim, action_dim)
        self.optimizer = optim.Adam(self.policy.parameters(), lr=3e-4)

        self.clip_epsilon = 0.2
        self.gamma = 0.99
        self.gae_lambda = 0.95
        self.epochs = 10
        self.batch_size = 64

    def compute_gae(self, rewards, values, dones):
        advantages = []
        gae = 0

        for t in reversed(range(len(rewards))):
            if t == len(rewards) - 1:
                next_value = 0
            else:
                next_value = values[t + 1]

            delta = rewards[t] + self.gamma * next_value * (1 - dones[t]) - values[t]
            gae = delta + self.gamma * self.gae_lambda * (1 - dones[t]) * gae
            advantages.insert(0, gae)

        return torch.FloatTensor(advantages)

    def update(self, states, actions, old_log_probs, returns, advantages):
        for _ in range(self.epochs):
            # Get current policy outputs
            action_mean, action_std, values = self.policy(states)
            dist = Normal(action_mean, action_std)
            log_probs = dist.log_prob(actions).sum(dim=-1)
            entropy = dist.entropy().sum(dim=-1).mean()

            # Compute ratios
            ratios = torch.exp(log_probs - old_log_probs)

            # Clipped surrogate objective
            surr1 = ratios * advantages
            surr2 = torch.clamp(ratios, 1 - self.clip_epsilon, 1 + self.clip_epsilon) * advantages
            actor_loss = -torch.min(surr1, surr2).mean()

            # Value loss
            critic_loss = nn.MSELoss()(values.squeeze(), returns)

            # Total loss
            loss = actor_loss + 0.5 * critic_loss - 0.01 * entropy

            # Optimize
            self.optimizer.zero_grad()
            loss.backward()
            nn.utils.clip_grad_norm_(self.policy.parameters(), 0.5)
            self.optimizer.step()

# Training with PPO
agent = PPO(state_dim=17, action_dim=6)  # For robot arm

for iteration in range(1000):
    states, actions, rewards, dones, log_probs, values = [], [], [], [], [], []

    # Collect trajectories
    state = env.reset()
    for step in range(2048):
        state_tensor = torch.FloatTensor(state).unsqueeze(0)
        action, log_prob, value = agent.policy.get_action(state_tensor)

        next_state, reward, done, _ = env.step(action.detach().numpy()[0])

        states.append(state)
        actions.append(action.detach().numpy()[0])
        rewards.append(reward)
        dones.append(done)
        log_probs.append(log_prob.item())
        values.append(value.item())

        state = next_state if not done else env.reset()

    # Convert to tensors
    states = torch.FloatTensor(states)
    actions = torch.FloatTensor(actions)
    old_log_probs = torch.FloatTensor(log_probs)

    # Compute advantages and returns
    advantages = agent.compute_gae(rewards, values, dones)
    returns = advantages + torch.FloatTensor(values)

    # Normalize advantages
    advantages = (advantages - advantages.mean()) / (advantages.std() + 1e-8)

    # Update policy
    agent.update(states, actions, old_log_probs, returns, advantages)

    print(f"Iteration {iteration}, Avg Reward: {sum(rewards) / len(rewards):.2f}")

class SAC:
    def __init__(self, state_dim, action_dim):
        self.actor = Actor(state_dim, action_dim)
        self.critic1 = Critic(state_dim, action_dim)
        self.critic2 = Critic(state_dim, action_dim)

        self.target_critic1 = Critic(state_dim, action_dim)
        self.target_critic2 = Critic(state_dim, action_dim)

        self.alpha = 0.2  # Temperature parameter
        self.gamma = 0.99
        self.tau = 0.005  # Soft update

    def select_action(self, state, deterministic=False):
        with torch.no_grad():
            if deterministic:
                return self.actor.get_mean_action(state)
            return self.actor.sample_action(state)

Reward Shaping for Robotics​

Manipulation Tasks​

def compute_manipulation_reward(robot_state, object_state, target_state):
    # Distance to object
    dist_to_object = np.linalg.norm(robot_state['gripper_pos'] - object_state['pos'])
    reach_reward = -dist_to_object

    # Grasping reward
    is_grasped = check_grasp(robot_state, object_state)
    grasp_reward = 10.0 if is_grasped else 0.0

    # Object to target distance
    dist_to_target = np.linalg.norm(object_state['pos'] - target_state['pos'])
    place_reward = -dist_to_target if is_grasped else 0.0

    # Success bonus
    success = dist_to_target < 0.05 and is_grasped
    success_reward = 100.0 if success else 0.0

    # Total reward
    reward = reach_reward + grasp_reward + place_reward + success_reward

    # Penalties
    if robot_state['collision']:
        reward -= 5.0

    return reward

Locomotion Tasks​

def compute_locomotion_reward(robot_state, target_velocity):
    # Forward velocity reward
    velocity_reward = robot_state['forward_velocity'] * 1.0

    # Stability penalty (penalize falling)
    height_penalty = -5.0 if robot_state['torso_height'] < 0.3 else 0.0

    # Energy penalty
    energy_penalty = -0.01 * np.sum(np.square(robot_state['joint_torques']))

    # Orientation penalty (keep upright)
    orientation_penalty = -0.5 * abs(robot_state['roll']) - 0.5 * abs(robot_state['pitch'])

    # Deviation from target velocity
    velocity_error = abs(robot_state['forward_velocity'] - target_velocity)
    tracking_reward = -velocity_error

    return velocity_reward + height_penalty + energy_penalty + orientation_penalty + tracking_reward

Sim-to-Real Transfer​

Domain Randomization​

class DomainRandomization:
    def __init__(self):
        self.params = {
            'mass': (0.8, 1.2),
            'friction': (0.5, 1.5),
            'damping': (0.8, 1.2),
            'motor_strength': (0.9, 1.1),
            'latency': (0, 0.05)  # seconds
        }

    def randomize(self, env):
        # Randomize physics
        mass_scale = np.random.uniform(*self.params['mass'])
        env.set_mass_scale(mass_scale)

        friction = np.random.uniform(*self.params['friction'])
        env.set_friction(friction)

        # Randomize actuation
        motor_strength = np.random.uniform(*self.params['motor_strength'])
        env.set_motor_strength(motor_strength)

        # Add latency
        latency = np.random.uniform(*self.params['latency'])
        env.set_action_latency(latency)

        return env

Residual RL​

class ResidualPolicy:
    def __init__(self, base_controller, rl_policy):
        self.base = base_controller
        self.rl = rl_policy

    def get_action(self, state):
        # Base controller (e.g., PD controller)
        base_action = self.base.compute(state)

        # RL residual
        residual = self.rl.get_action(state)

        # Combined action
        return base_action + 0.1 * residual  # Scale residual

Key Takeaways​

  • RL enables robots to learn complex behaviors through trial and error
  • PPO is state-of-the-art for continuous control robotics tasks
  • Reward shaping is critical for successful learning
  • Sim-to-real transfer requires domain randomization and physics variation
  • Isaac Gym enables massively parallel training (10-1000x speedup)
  • Stable-Baselines3 provides easy-to-use RL implementations
  • Curriculum learning helps tackle complex multi-stage tasks
  • Residual RL combines classical control with learned policies
